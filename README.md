## 微信小程序demo

### WXML 模板

#### 简介

使用上跟HTML语法类似，区别是：

 * 标签必须都是闭合的
 * 属性大小写敏感
 
#### 数据绑定 {{}}
 
 完成界面的实时更新。
 
 xxx.js 文件
 
* 页面初始数据

	``` js
	<!--初始化数据 time-->
	Page({
		data: {
			time: (new Date()).toString()
		}
	})
	```
	
	``` wxml
	<!--数据绑定-->
	<text>{{time}}</text>
	```
注： 属性值的绑定必须要包裹在双引号中。

#### 逻辑语法

{{}} 中可以进行简单的逻辑运算。注意：要在{{}}内部。

* 三元运算
* 算术运算
* 字符串变量拼接
* 数组
